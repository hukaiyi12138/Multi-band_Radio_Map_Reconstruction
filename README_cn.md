# DUSPF-RME

**DUSPF-RME** 是一种用于**无线电地图估计**（Radio Map Estimation）的**深度展开网络**（Deep Unrolled Network），采用了 **SLF-PSD 分解**技术。该方法通过融合物理驱动的传播先验、SLF-PSD 分解以及基于 ADMM 的深度展开机制，实现了从稀疏测量数据中进行多频段无线电地图补全。

## 概述

DUSPF-RME 将多频段无线电地图估计建模为一个受 SLF-PSD 约束的张量补全问题。与依赖通用的张量核范数最小化不同，该方法利用一种感知发射源（emitter-aware）的低秩结构来表示无线电地图张量。具体而言，每个发射机由空间损耗场（SLF）和功率谱密度（PSD）向量进行建模，从而能够在极度稀疏的测量条件下实现高效的多频段无线电地图恢复。

该优化问题被展开为基于 ADMM 迭代过程的深度网络。这种设计结合了模型驱动优化的可解释性与深度神经网络强大的表征能力。

## 关键特性

### SLF-PSD 分解

DUSPF-RME 将多频段无线电地图分解为各发射机的 SLF 与 PSD 向量之间的秩-(R) 外积形式。这种结构既捕捉了频域的低秩相关性，又避免了 Tucker 核范数最小化所需的奇异值阈值处理操作。

### 联合 PSD 初始化

PSD 系数采用 Tikhonov 正则化的联合最小二乘法进行初始化。该初始化过程利用了观测支撑集（observed support）上各发射机间的耦合关系，为后续的 ADMM 迭代提供了稳定的起点。

### 物理驱动的传播先验

对于每个发射机，DUSPF-RME 通过结合自由空间路径损耗与射线追踪几何阴影效应，构建了一个可学习的物理驱动传播先验。路径损耗指数和阴影衰减系数与网络进行端到端联合优化。由于该先验仅依赖于建筑地图和发射机位置，即使在可用测量数据极度稀疏的情况下，它仍能提供有效的信息。

### 物理感知 U-Net 近端算子

SLF 的近端映射（proximal mapping）通过一个可学习的 U-Net 模块来实现。 U-Net 通过在传播先验的基础上预测残差校正，从而优化 SLF（信号场）估计。该网络利用具有物理意义的各发射源特征作为输入条件，这些特征包括距离图、射线投射阴影场、建筑物分布图以及传播先验信息。

与浅层局部卷积模块相比，U-Net 提供了更大的有效感受野，这有助于在极低采样率下将稀疏观测信息传播至整个空间网格。

## 实验结果

在 BART-Lab 无线电地图数据集上的实验表明，在 1% 到 15% 的采样率范围内，DUSPF-RME 的表现始终优于现有方法。与 RadioUNet 相比，DUSPF-RME 实现了更高的峰值信噪比（PSNR）和更准确的信号中断检测，证明了其在稀疏多频段无线电地图估计任务中的有效性。
